Como colocar um favicon em um site?

Para adicionar um favicon é simples:

1. Baixe uma imagem (de preferência transparente).

2. Converta para ícone (.ico) usando o site favicon.io.

3. Coloque a imagem na mesma pasta do site em desenvolvimento.

4. Na tag <head>, selecione o conteúdo dentro de <href=""> do HTML e apague.

5. Em seguida, pressione as teclas ctrl + espaço e selecione o arquivo desejado.

Tudo pronto. Agora é só carregar o site.
